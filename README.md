## 🛠️ Requirements

- GCC Compiler
- Code::Blocks, Dev-C++, Visual Studio Code, or any C IDE

## ▶️ Compile and Run

Compile:

```bash
gcc filename.c -o filename
```

Run:

```bash
./filename
```

On Windows:

```bash
filename.exe
```
